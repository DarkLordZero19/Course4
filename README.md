# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание панели фильтров: Практическая Работа, в котором надо Применить техники React для создания производительного и чистого кода. Научиться предотвращать лишние ре-рендеры с помощью React.memo, мемоизировать «тяжелые» вычисления с useMemo, выносить переиспользуемую логику в кастомные хуки и использовать продвинутые паттерны условного рендеринга.

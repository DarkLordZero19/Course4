# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание приложения «Корзина товаров»: Практическое задание в котором нужно комплексно применить все изученные концепции React для создания полноценного интерактивного приложения. Научиться управлять сложным состоянием (массив объектов), выстраивать логику взаимодействия между компонентами через props и функции обратного вызова, и использовать useEffect для вычисляемых побочных эффектов.

# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание автосохраняемого поля для заметок: Задание в котором нужно Самостоятельно применить хук useEffect для синхронизации состояния компонента с внешним хранилищем (localStorage). Научиться использовать два разных сценария useEffect в одном компоненте: для сохранения данных при их изменении и для загрузки данных при первом рендере.

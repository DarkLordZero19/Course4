# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание Todo List с помощью Context API: Задание, в котором нужно самостоятельно применить React Context API для управления глобальным состоянием приложения. Научиться создавать кастомный провайдер для инкапсуляции всей логики, а также потреблять данные и функции из контекста в разных, не связанных друг с другом, компонентах, чтобы полностью избавиться от «prop drilling».

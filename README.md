# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание приложения «База данных фильмов»: Практическая работа, в котором нужно комплексно применить все изученные концепции React (компоненты, props, state, useEffect, роутинг, Context API) для создания полноценного интерактивного SPA. Научиться выстраивать компонентную архитектуру, управлять локальным и глобальным состоянием, симулировать асинхронные операции и создавать качественный пользовательский опыт.

# Course4
Репозиторий для обучения REACT в четвертом курсе.
- Создание страницы настроек с использованием Context API: Практическое задание в котором нужно Научиться решать проблему «пробрасывания пропсов» (prop drilling) с помощью React Context API. Освоить полный цикл работы с контекстом: создание, предоставление (Provider) и потребление (useContext). Научиться создавать кастомные провайдеры для инкапсуляции логики и использовать несколько контекстов в одном приложении.
